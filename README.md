## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

유튜브, 트위치(치지직), 아프리카tv 통합 검색 및 분석 프로젝트


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
